🏥 MedWatch Setup Guide

Complete Setup & Admin Account Creation

📌 Important: Follow these steps in order to set up your MedWatch application.

1 Create Admin Account

Before you can access the admin panel, you need to create an admin account.

Method 1: Using Batch File (Easiest)

Double-click on: setup-admin.bat in your project folder

Method 2: Using PowerShell

cd server
node quick-admin-setup.js

Method 3: Manual Command

cd server
node create-admin.js

🔑 Default Admin Credentials

Username: admin
Password: admin123
Role: Admin
Hospital: MY Hospital

⚠️ Change this password after your first login!

2 Start the Application

Double-click on: start-app.bat in your project folder

This will automatically:

  • ✅ Start the Backend Server (Port 5000)
  • ✅ Start the Frontend Dev Server (Port 4002)
  • ✅ Open your browser to the application

3 Login to Admin Panel

  1. Go to: http://localhost:4002/login
  2. Click on the "Admin/Hospital" tab
  3. Enter username: admin
  4. Enter password: admin123
  5. Click "Sign In"
🚀 Open Login Page

4 Create Additional Accounts

For Doctor/Nurse/Pharmacist accounts:

  1. Go to: http://localhost:4002/signup
  2. Fill in the registration form
  3. Select role (Doctor, Nurse, or Pharmacist)
  4. Select hospital from dropdown
  5. Click "Create Account"
Note: Admin accounts CANNOT be created through the signup page for security reasons. Only the script above can create admin accounts.
📝 Open Signup Page

🔧 Troubleshooting

Problem: "Connection failed" or ports in use

Solution:

# Kill processes on ports
Get-Process -Id (Get-NetTCPConnection -LocalPort 5000).OwningProcess | Stop-Process -Force
Get-Process -Id (Get-NetTCPConnection -LocalPort 4002).OwningProcess | Stop-Process -Force

Problem: MongoDB not running

Solution (PowerShell as Admin):

Start-Service MongoDB

Problem: Teal colors not showing

Solution: Press Ctrl + Shift + R to hard refresh browser

Problem: Invalid credentials

Solution: Run the admin creation script again to reset password
